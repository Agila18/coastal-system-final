# Hydro Hub Frontend

A stunning full-screen hero section for Hydro Hub - Tamil Nadu's coastal risk prediction platform.

## Features

- **Full-screen Hero Section** with coastal-themed gradient background
- **Responsive Design** optimized for all devices
- **Smooth Animations** including fade-in, slide-up, and floating effects
- **Glassmorphism UI** for stats cards with backdrop blur
- **Tailwind CSS** for utility-first styling
- **Custom Color Palette** with teal, cyan, and navy theme

## Tech Stack

- React 18
- Tailwind CSS 3
- Pure CSS animations (no external animation libraries)

## Installation & Setup

For full-stack setup instructions (including Backend and Database), please refer to the **[Main README](../README.md)**.

### Quick Start (Frontend Only)

```bash
npm install
npm start
```

## Build for Production

```bash
npm run build
```

## Component Structure

- **HeroSection.jsx** - Main hero component with all UI elements
- **HeroSection.css** - Custom animations and responsive styles
- **tailwind.config.js** - Tailwind configuration with Hydro Hub theme
